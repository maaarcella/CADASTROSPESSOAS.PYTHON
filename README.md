# 📋 Sistema de Cadastro de Contatos em Python

## Descrição

Este projeto consiste em um sistema simples de cadastro de contatos desenvolvido em Python para execução no terminal. A aplicação permite registrar informações de usuários, como nome, e-mail e telefone, além de listar os contatos cadastrados e realizar operações de gerenciamento.

O sistema foi criado com o objetivo de praticar conceitos fundamentais da programação, utilizando listas para armazenar dados e estruturas de controle para gerenciar as funcionalidades do programa.

## Funcionalidades

* Cadastro de contatos;
* Armazenamento de nome, e-mail e telefone;
* Listagem de todos os contatos cadastrados;
* Menu interativo no terminal;
* Encerramento do sistema por opção do usuário;
* Estrutura preparada para exclusão de registros.

## Tecnologias Utilizadas

* Python 3
* Biblioteca `os`
* Biblioteca `time`

## Objetivo do Projeto

O principal objetivo deste projeto é desenvolver habilidades relacionadas à programação básica em Python, incluindo manipulação de listas, entrada e saída de dados, estruturas condicionais e laços de repetição.

Além disso, o sistema simula uma pequena agenda de contatos, permitindo ao usuário armazenar e consultar informações de forma simples e organizada.

## Como Funciona

Ao iniciar o programa, o usuário visualiza um menu com as seguintes opções:

1. Cadastrar contato;
2. Listar contatos cadastrados;
3. Excluir contato;
4. Sair do sistema.

Os dados são armazenados em listas durante a execução do programa e podem ser consultados a qualquer momento através da opção de listagem.

## Aprendizados

Durante o desenvolvimento deste projeto foram aplicados conceitos como:

* Variáveis;
* Listas;
* Estruturas condicionais (`if`, `elif`);
* Laços de repetição (`while`);
* Função `zip()`;
* Entrada e saída de dados;
* Organização de programas em Python.
